- POST `/api/auth/register`
- POST `/api/auth/login`
- GET `/api/auth/me`
- PUT `/api/profile`
- GET `/api/lessons`
- GET `/api/lessons/day/{dayNumber}`

- GET `/api/practice/history`
- GET `/api/notebook`

- GET `/api/reports/summary`
- GET `/api/reports/history`
- POST `/api/reports/history`
